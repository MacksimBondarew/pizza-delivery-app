## 📌 **Overview**

**AI Personal Tutor** is a full-stack AI-powered platform that transforms **PDF learning materials into interactive study experiences**.

Users can:
- upload documents  
- generate quizzes automatically  
- chat with AI about content  
- track learning progress  

Built with **Next.js App Router**, **Supabase**, and **OpenAI API**, the app focuses on **performance, scalability, and modern UX**.

---

## 📋 **Core Technologies**

- 🏗 **Frontend**: `Next.js`, `React`, `TailwindCSS`
- ⚡ **Backend**: `Next.js Route Handlers`
- 🗃 **Database**: `Supabase Postgres`
- 🔐 **Authentication**: `Supabase Auth`
- ☁️ **Storage**: `Supabase Storage (PDFs)`
- 🧠 **AI**: `OpenAI API`
- 🎯 **State Management**: `@tanstack/react-query`
- 📦 **Forms**: `react-hook-form`, `zod`
- 🔧 **UI/Utils**: `Radix UI`, `sonner`, `clsx`, `cva`

---

## 🚀 **Features**

✔️ Upload and manage PDF documents  
✔️ AI-generated quizzes from study materials  
✔️ Context-aware chat with documents  
✔️ User authentication and protected routes  
✔️ Progress tracking and analytics  
✔️ Responsive UI (mobile + desktop)
